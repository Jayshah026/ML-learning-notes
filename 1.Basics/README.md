## 2. Unsupervised learning

```Python 

Model is trained on unlabeled data — only input is provided. The model finds hidden structure/patterns on its own, without a known correct output.

There are 4 types of unsupervised learning. 

1. Clustering
2. Dimensionality Reduction
3. Anomaly Detection
4. Association

```
---

### 1. Clustering 

```Python

- Groups data points based on similarity in their features (no predefined labels).

Example: customer segmentation based on purchase behavior.
Common algorithm: K-Means.

```
---

### 2. Dimensionality Reduction

```Python

- Reduces the number of input features while preserving as much information/variance as possible. Useful when multiple features are correlated/redundant.

Benefits: less storage, faster training, fights the "curse of dimensionality."
Common algorithm: PCA (Principal Component Analysis).

```
---

### 3. Anomaly Detection

```Python

- Identifies data points that deviate(Different data from the original data) significantly from the normal/expected pattern in the data.

Example: fraud detection, network intrusion detection.

```
---

### 4. Association

```Python

- Discovers rules describing co-occurrence relationships between variables.

Example (market basket analysis): customers who buy bread also tend to buy butter.

```
---

## 3.  Semi-Supervised Learning

```Python

- Uses a small amount of labeled data combined with a large amount of unlabeled data during training.

- Means here we will labeled some data and then when comes to training part the rest of the data which is not lebeled will be trained by this lebeled data.

- Here alon supervised learning is costly and that's why here we have small amount of lebeled data by which we can train the other data which is not lebeled data. 

```
---

## 4. Reinforcement Learning

```Python

- An agent learns by interacting with an environment, taking actions, and receiving rewards or penalties, aiming to maximize cumulative reward over time.

Ex : Self driving car
```

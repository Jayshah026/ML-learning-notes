# What is machine learning : 

- So a machine learning is a method in which we provide data and output so that the machine can train on that data, After it trains on that data and we give a new
  input similar to the data it was trained, it will automatically understand the data and will give us a output.

- So in simple terms it trains on data and make an algorithm so that in future, if we have any questions similar to the data it was trained we can have an answer
  instantly and accurately.

# AI - ML - DL : 

## AI (Artificial Intelligence) : 
- So it is a method in which it try to recognise the patterns. Like we humans can classify things just by looking at it the AI is trying to do the same in which it
  try to understand the patterns and gives us an appropriate answer. More or less it try to become like a human.

Ex : Classifying the dog from the cat,dog and elephant. 

## ML (Machine Learning) :
- So in this method we provide data and output togather so that the model or machine can train on that data. After train on that data when we are asking questions out
  of that data it will still recognise the pattern and accordingly it will give us an output.

Ex : Train a model on the data to do 2 numbers sum. Now after that if we give any 2 numbers to get the sum of it, it will give us an appropriate answer.

## DL (Deep Learning) : 
- So it is just like the machine learning method but in this the architecture is copied from the human brain. Means the architecture is complex but accurate + it is
  mainly used for the big data. Where the ML is used for medium or small data for more accuracy.

Ex : Speech Recognition.

# Types of machine learning :

1. Supervised learning
2. Unsupervised learning
3. Semisupervised learning
4. Reinforcement Learning

## 1. Supervised Learning : 

- Imagine you have the data which is including both the input and output.
- Now machine is try to learn and understad the relation between that input and output so that when you give a new input to that machine it can give you the output based on that learning.
- This learning is called Supervised learning.

### There are two types of Supervised learning : 

1. Regression
2. Classification

#### 1. Regression : 

- It is a part of machine learning.
- When the output is Numerical then it's called Regression.

Ex : --- input : CGPA and IQ --- And --- Output : Package ---

Now based on the CGPA and the IQ machine learning will pridict the package of the student. Now package will be a numerical output which will be considerd as a Regressesion. 

#### 2. Classification : 

- It is also a part of machine learning.
- When output can be classified or categorized it is fall in this category. 

Ex : --- Input : Weather forecast --- And --- Output : Condition of the weather ---

- Now the output can be anything like sunny, rainy and so on.. Which is buch of category and we are picking one of them as per the data is called Classification.
